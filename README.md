# RythmML

## Install python-rtmidi

### Linux
Intall first those packages :
- libasound2-dev
- libjack-jackd2-dev

```shell script
apt install libasound2-dev libjack-jackd2-dev
pip3 install python-rtmidi
```

### Windows

```
pip install python-rtmidi
```

If Cython is needed:
```
pip install Cython
```
